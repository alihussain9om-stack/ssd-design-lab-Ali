## Step 6 – Evaluate UML Tools

| Criterion | Tool A: diagrams.net | Tool B: PlantUML Web |
|---------|----------------------|----------------------|
| Learning curve | Very easy; drag-and-drop interface | Requires UML syntax; fast after basics |
| Version control friendliness | `.drawio` can be diff’d as XML; PNG for review | Plain text diff-friendly; PNG generated from source |
| Validation support | Manual visual checks only | Syntax validation; basic lint via editor |
| Collaboration | Works well with Google Drive / OneDrive | Text-based PR reviews are easy |
| Export formats | PNG, SVG, PDF | PNG (regenerated from source) |
| Cost | Free | Free |
